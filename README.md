# C-Note
一个日记类微信小程序, 可以上传图片， 修改背景

### 介绍
1. 整个小程序无后台
2. 数据存储方案使用的是 [leancloud](https://leancloud.cn/)
3. 天气和位置查询 使用的是 [和风天气](https://www.heweather.com/)
4. 所有界面UI使用纯wxss完成.
